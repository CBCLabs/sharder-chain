# Sharder Client UI #

## Compile and run in npm
1. run `npm i`
2. run `npm run generate_theme`
3. run `npm run dev`
4. access URL: [http://localhost:4000](http://localhost:4000)

## Build and run in Sharder-Client
1. run `npm i`
2. run `npm run generate_theme`
3. run `npm run build`
4. startup Sharder
5. access URL: [http://localhost:8215](http://localhost:8215)
